# Unity Inventory System

## 📦 개요
이 프로젝트는 **Unity**로 제작된 간단한 **인벤토리 시스템**입니다.  
아이템을 **강제로 생성하여 슬롯에 배치**하고, **장착 시 플레이어의 스탯이 상승**하는 구조를 가지고 있습니다.

---

## ✅ 구현된 기능

- 아이템 자동 생성 및 인벤토리 등록
- 인벤토리 UI 표시 (GridLayout 기반)
- 아이템 선택 시 상세 정보 출력
- 장착 및 해제 기능
  - 장착 시 해당 아이템의 스탯이 플레이어에게 반영됨
